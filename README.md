# 我的世界简易聊天翻译

## ---用途---
1、将聊天框中的内容翻译成中文
2、将需要发送的中文翻译成英文发送

## ---使用说明---
1下载前置模组:模组菜单Mod Menu
2.在模组菜单中使用本地翻译或配置秘钥

## ---目前支持的游戏版本和模组加载器---
### Java Edition (1.21.8，1.21.11) + Fabric
## ---未来的更新---
### 1游戏内更改配置(已完成)
### 2支持将输入内容翻译成英文输出(已完成)
### 3支持多模组加载器
### 4翻译服务器物品名称，物品简介和Tab名称栏

# Minecraft Simple Chat Translation
## ---Purpose---
1. Translate the content in the chat box into Chinese

2. Translate the Chinese text to be sent into English and send it
##   ---Instructions---
1. Download the prerequisite module: Mod Menu
2. Use local translation or configuration key in the module menu

## ---Currently supported game versions and mod loaders---

### Java Edition (1.21.8，1.21.11) + Fabric

## ---Future Updates---

### 1 In-game configuration changes (completed)
### 2 Support translating input content into English output (completed)
### 3 Support for multiple mod loader
### 4 Translate server item names, item descriptions, and tab names
