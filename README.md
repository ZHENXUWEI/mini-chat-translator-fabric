# 我的世界简易聊天翻译
## ---用途---
### 将聊天框中的内容翻译成中文
## ---使用说明---
### 1安装模组后必须运行一次游戏然后退出
### 2打开游戏文件夹
### 3打开目录下的Config文件夹
### 4使用记事本打开mini-chat-translator.properties
### 5填写百度翻译的APPID和秘钥
## ---目前支持的游戏版本和模组加载器---
### Java Edition (1.21.11) + Fabric
## ---未来的更新---

### 2支持将输入内容翻译成英文输出
### 3支持多平台
